# Review & Rating Service
Product reviews and ratings.

## Endpoints
- `POST /reviews` тАФ Submit review
- `GET /reviews/:productId` тАФ Get product reviews
- `GET /reviews/average/:productId` тАФ Average rating & distribution
- `PUT /reviews/:reviewId/helpful` тАФ Mark helpful
- `DELETE /reviews/:reviewId` тАФ Delete review
- `GET /health` тАФ Health check
